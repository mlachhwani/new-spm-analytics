# RTIS Driving Behaviour Analysis

This repository contains a railway-grade analysis system for evaluating
locomotive driving behaviour using RTIS data.

## Key Concepts
- Section-based signal analysis
- Signal-inferred violations
- Train-type-specific speed rules
- Structured PDF reporting
- Google Drive archival

## Data Inputs
- RTIS file (CSV)
- Section-wise signal datasets
- Crew master
- Route reference (informational only)

## Output
- Interactive graphs
- Multi-page PDF reports
